# Task Manager CLI

Application de gestion de tâches en Dart.

## Installation

dart pub get

## Lancer

dart run main.dart

## Lancer les tests

dart test

## Fonctionnalités

- Ajouter une tâche
- Lister les tâches
- Trier par priorité
- Trier par date
- Marquer comme terminée
- Supprimer
- Persistance JSON

## Architecture

- Classes abstraites
- Héritage
- Interface
- Génériques
- Exceptions personnalisées
- Tests unitaires
